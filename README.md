# 📊 Financial Performance Dashboard – Power BI

## 📌 Project Overview
This project presents a **Financial Performance Dashboard** built using Power BI.
It provides insights into business performance using key financial metrics and interactive visualizations.

## 🎯 Objectives
* Analyze financial data effectively
* Track revenue, profit, and expenses
* Identify trends and business performance
* Support data-driven decision making

## 🛠 Tools & Technologies Used
* Power BI
* Microsoft Excel / CSV
* Data Cleaning & Transformation

## 📊 Key Features
* 📈 Revenue & Profit Analysis
* 📉 Expense Tracking
* 📅 Time-based Trends
* 📊 KPI Indicators
* 🔍 Interactive Filters & Slicers
